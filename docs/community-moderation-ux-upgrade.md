# ServerCore v24.1 Community & Moderation UX Upgrade

Пакет улучшает эксплуатацию сообщества и модерации без смены основной версии проекта.

## Что добавлено

- Тикеты 2.0: статус, приоритет, назначенный модератор, комментарий модерации.
- Расширенная карточка пользователя в веб-панели: экономика, покупки, тикеты, дела, заявки, ивенты.
- Журнал входов в веб-панель: успешные и неуспешные попытки.
- AutoMod Rules 2.0: отдельные действия для ссылок, капса, спама, упоминаний и запрещенных слов.
- Онбординг с наградой: после выполнения чек-листа пользователь получает монеты.
- История покупок магазина и связка с товаром дня/скидками.
- VPS migration helper: команда `/vps-check` и страница `/vps`.

## Команды после обновления

После redeploy нужно выполнить:

```bash
npm run deploy
```

Новые/обновленные команды:

- `/ticket` — добавлен параметр приоритета.
- `/vps-check` — проверка готовности к будущему переносу на VPS.

## Веб-панель

Новые или улучшенные страницы:

- `/tickets`
- `/user/:id`
- `/web-logins`
- `/purchase-history`
- `/automod-rules`
- `/vps`
- `/onboarding`
