# v24.1 User Web Panel & Auth Upgrade

Добавлена двухуровневая авторизация веб-панели:

- **Admin**: логин `Admin` + пароль `WEB_PASSWORD` из переменных окружения.
- **User**: Discord ID + пароль/одноразовый код, созданный через Discord-команду `/webaccount`.

## Команды Discord

- `/webaccount status` — статус пользовательского веб-аккаунта.
- `/webaccount password value:<пароль>` — создать или изменить пароль пользователя.
- `/webaccount login-code` — получить одноразовый код входа на 10 минут.
- `/webaccount disable` — отключить свой пользовательский веб-аккаунт.

## Пользовательская веб-панель

После входа в режиме **Пользователь** доступны только личные разделы:

- `/user-panel` — сводка.
- `/my-profile` — профиль.
- `/my-balance` — баланс и операции.
- `/my-inventory` — инвентарь.
- `/my-tickets` — тикеты пользователя.
- `/my-events` — активности пользователя.
- `/my-purchases` — покупки.
- `/account` — данные веб-аккаунта и последние входы.

Пользовательские сессии отделены от административных. Пользователь не может открыть административные страницы — при попытке будет перенаправлен в `/user-panel`.

## Безопасность

Пароли пользователей не хранятся в открытом виде. Используется PBKDF2-хэш с солью. Одноразовый код сбрасывается после успешного входа.
