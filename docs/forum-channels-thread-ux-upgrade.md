# v24.1 Forum Channels & Thread UX Upgrade

## Что изменилось

Добавлена поддержка forum-каналов и панелей создания тем. Часть каналов, где удобнее вести отдельные обсуждения, теперь создается как Forum Channel.

## Каналы, которые переводятся в forum

- 🧵・темы-участников
- ❓・вопросы-и-помощь
- 📷・скрины-и-фото
- 💡・предложения
- 🎮・поиск-команды
- 🛠・помощь-с-кодом
- 🚀・проекты-участников
- 📨・заявки
- 🚨・жалобы
- 📁・разборы-и-кейсы

## Что остается обычными текстовыми каналами

Правила, объявления, навигация, команды, магазин, панель модерации, логи и другие панельные каналы остаются текстовыми.

## Важное поведение setup

Discord не позволяет менять тип существующего канала Text → Forum. Поэтому `npm run setup` при конфликте переименует старый текстовый канал в `название-архив`, а затем создаст новый forum-канал с исходным названием.

## Команда

- `/threadpanel open` — открыть панель создания темы для себя.
- `/threadpanel post` — опубликовать панель в канал, доступно администраторам.

## Веб-панель

Новая страница:

- `/forum-channels` — показывает найденные forum-каналы, теги и рекомендуемые разделы.

Также панель тем доступна на странице `/panels`.
