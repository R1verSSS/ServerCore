# v24.1 Bot Quick Menu Restore

Восстановлена панель **🤖 Быстрое меню бота** для канала `🤖・команды-бота` после добавления forum/thread структуры.

Панель доступна в веб-панели `/panels` и `/quick-actions`, а также публикуется через `npm run setup`.

Кнопки панели выполняют действия сразу: профиль, ранг, daily, баланс, история баланса, инвентарь, магазин, мини-игры, квест, тикет, достижения, Battle Pass, веб-панель и справка.
