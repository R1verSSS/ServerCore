# v24.2 Channel Structure & Navigation Cleanup

Обновление упрощает навигацию и безопасно переносит существующие каналы без создания дублей.

## Новая структура

- `📌 ВАЖНОЕ` — правила, объявления, навигация, роли и новости.
- `💬 ОБЩЕНИЕ` — общий чат, темы участников, фото, предложения и мемы.
- `🤖 БОТ И АКТИВНОСТИ` — команды бота, музыка, мини-игры, турниры и ивенты.
- `🎮 ИГРЫ` — поиск напарников и кланы.
- `🛒 МАГАЗИН` — витрина, инвентарь, подарки и Premium.
- `💻 ПРОЕКТЫ И IT` — программирование, помощь с кодом, материалы и проекты.
- `🎫 ПОДДЕРЖКА` — вопросы, создание тикета и частые вопросы.
- `🔊 ГОЛОСОВЫЕ` — постоянные и временные голосовые комнаты.
- `🛡 МОДЕРАЦИЯ` — закрытые рабочие каналы команды сервера.

## Переименования

- `🎮・поиск-команды` → `🎮・поиск-напарников`.
- текстовый `🎵・музыка` → `🎵・музыка-бот`.
- голосовой `🎵・музыка` → `🎵・музыкальная-комната`.
- `📚・команды` → `📚・частые-вопросы`.
- `📨・заявки` → `📥・заявки`.
- `🛡・лог-модерации` → `📋・лог-модерации`.
- `📂・тикеты` → `📁・тикеты`.
- `📘・команды-модерации` → `🤖・команды-модерации`.

## Безопасная миграция

`npm run setup` теперь ищет каналы по текущим и старым именам, затем:

1. переименовывает существующий канал;
2. переносит его в новую категорию;
3. обновляет тему канала;
4. сохраняет сообщения и историю;
5. создаёт новый канал только тогда, когда подходящего канала действительно нет;
6. выставляет единый порядок категорий и каналов.

Если тип канала необходимо изменить с Text на Forum, Discord не позволяет сделать это напрямую. В таком случае старый канал получает суффикс `-архив`, а новый Forum создаётся рядом.

## Тикеты

Публичная категория называется `🎫 ПОДДЕРЖКА`. Приватные активные тикеты теперь рекомендуется хранить отдельно в категории `🎫 АКТИВНЫЕ ТИКЕТЫ`.

В `.env` можно указать:

```env
TICKET_ACTIVE_CATEGORY_NAME=🎫 АКТИВНЫЕ ТИКЕТЫ
```

## Установка

```bash
npm run check
npm run setup
npm run deploy
```

После `setup` проверьте права категории `🎫 ПОДДЕРЖКА` и закреплённую панель в `🎫・создать-тикет`.
