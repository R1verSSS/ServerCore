# Portfolio checklist

Перед публикацией проекта на GitHub проверь:

- [ ] В репозитории нет `.env`.
- [ ] В репозитории нет `.git` из локального архива.
- [ ] В репозитории нет `data/database.sqlite`, `data/database.json`, backup/export/log files.
- [ ] Заполнен GitHub About.
- [ ] Добавлены topics: `discord-bot`, `nodejs`, `express`, `sqlite`, `docker`.
- [ ] В README есть описание продукта, запуск, Docker, архитектура, модули, roadmap.
- [ ] Добавлены скриншоты в `docs/assets/screenshots/`.
- [ ] Выполнены проверки `npm run check`, `npm run env:check`, `docker build`.
- [ ] Есть понятное описание проблем, которые решались.
- [ ] В резюме добавлена краткая формулировка из `docs/PORTFOLIO.md`.
