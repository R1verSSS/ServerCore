# ServerCore: музыка и защита информационных каналов

## Автоудаление сообщений не по теме

Бот автоматически удаляет обычные сообщения пользователей в информационных/панельных каналах, например:

- `📜・правила`
- `📢・объявления`
- `🧭・навигация`
- `✅・получить-роли`
- `📚・команды`
- `🤖・команды-бота`
- `🛍・витрина`
- `🎒・инвентарь`
- `🎁・подарки`
- `💎・premium`
- `📘・команды-модерации`

Модераторы и администраторы могут писать в этих каналах. Действия удаления пишутся в audit.

Дополнительные каналы можно добавить через переменную окружения:

```env
PROTECTED_TEXT_CHANNELS=канал-1,канал-2,канал-3
```

## Музыка в voice-комнатах

Добавлены команды:

```text
/music play url:<YouTube URL>
/music queue
/music pause
/music resume
/music skip
/music stop
/music leave
/musicpanel
```

Панель управления публикуется в текстовом канале `🎵・музыка` после `npm run setup`.

Порядок использования:

1. Пользователь заходит в voice-комнату.
2. В канале `🎵・музыка` нажимает `Включить YouTube`.
3. Вставляет ссылку YouTube video или playlist.
4. Управляет музыкой через кнопки.

Важно: используйте только контент, который разрешено воспроизводить на вашем сервере.
