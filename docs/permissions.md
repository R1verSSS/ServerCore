# Доступ и роли

В проект добавлен центральный контроль доступа для slash-команд, context-menu и важных кнопок.

## Уровни доступа

| Уровень | Назначение |
|---|---|
| Member | обычные пользовательские функции |
| VIP | расширенные пользовательские возможности |
| Helper | просмотр тикетов, заявок, предупреждений и истории |
| Moderator | базовая модерация: warn, mute, clear, cases |
| Admin | настройки, панели, AutoMod, backup/export, ивенты/турниры |
| Owner | критичные системные действия: restore, maintenance |

## Роли

Рекомендуемые роли сервера:

- 👑 Owner
- 🛡 Admin
- 👮 Moderator
- 🧰 Helper
- 💎 VIP

Страница веб-панели `/access` показывает матрицу доступа и проверку иерархии ролей.

## Context-menu

Добавлены действия по правому клику:

- Профиль
- Репутация
- История модерации
- Пожаловаться
- Удалить сообщение
- Warn за сообщение

## Важное

Даже если Discord скрывает команду через `default_member_permissions`, серверная проверка доступа остается внутри бота. Это защищает кнопки, select-menu и старые опубликованные панели.
