# Web Panel UX Upgrade для v24.1

Это обновление улучшает текущую веб-панель без добавления новой версии бота.

## Что улучшено

- Таблицы веб-панели: поиск, фильтры, сортировка и пагинация для участников и audit.
- Карточка участника: профиль, экономика, инвентарь, достижения, предупреждения, тикеты, заявки, дела и ивенты в одном месте.
- Мастер публикации Discord-панелей: главное меню, роли, мини-игры, модерация, заявки, тикеты и voice-инструкции.
- Безопасность веб-панели: лимит попыток входа, блокировка после нескольких ошибок, cookie-сессия на 8 часов и audit входов.
- Audit: подробные JSON-детали и фильтры по источнику/типу действия.
- Документация: старые README промежуточных патчей перенесены в `docs/history`.

## Новые страницы

- `/panels` — мастер публикации Discord-панелей.
- `/users` — улучшенная таблица участников.
- `/user/:id` — расширенная карточка участника.
- `/audit` — улучшенный audit-журнал.

## Настройки безопасности входа

Можно добавить в `.env`:

```env
WEB_LOGIN_LIMIT=5
WEB_LOGIN_LOCK_MINUTES=10
```

По умолчанию после 5 неверных попыток вход с IP блокируется на 10 минут.
