# ServerCore v24.1.18 — XP Level Roles & Action Menu Upgrade

## Что исправлено

### XP и роли за уровни

Теперь система XP автоматически нормализует уровень пользователя при начислении и при просмотре профиля/ранга.
Если пользователь достиг нужного уровня, бот синхронизирует роли:

- `Уровень 5 — Активный`
- `Уровень 10 — Свой`
- `Уровень 20 — Легенда`

Если роль отсутствует, бот попробует создать ее автоматически. Для этого у роли бота должно быть право `Manage Roles`, а роль бота должна быть выше выдаваемых ролей.

Дополнительно можно переопределить роли через env:

```env
LEVEL_ROLE_REWARDS=5:Уровень 5 — Активный,10:Уровень 10 — Свой,20:Уровень 20 — Легенда
```

### Меню выполняет действия сразу

Кнопки в `/menu open` и `/me` больше не просто подсказывают команды, а выполняют основные действия напрямую:

- профиль
- daily
- баланс
- инвентарь
- магазин
- мини-игры
- тикет
- достижения
- Battle Pass

### Ограничения кнопок по ролям

Для служебных кнопок усилен контроль доступа. Например, кнопка модераторской панели из меню доступна только модераторам и выше.
