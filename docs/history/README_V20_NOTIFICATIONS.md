# v20 Notifications & Reminders

Добавлено:

- `/remind create` — личное напоминание.
- `/remind list` — список активных напоминаний.
- `/remind cancel` — отмена напоминания.
- `/notify status` — просмотр системных уведомлений.
- `/notify set` — включение/выключение уведомлений.
- `/notify channel` — канал системных уведомлений.
- `/notify test` — ручной запуск проверки уведомлений.
- Веб-страница `/notifications`.
- Автоматическая проверка напоминаний каждые 30 секунд.
- Напоминания перед ивентами.
- Уведомления о новых турнирах.
- Уведомления о старых открытых тикетах.

Форматы времени для `/remind create`:

```text
10m
2h
1d
10.06.2026 20:00
2026-06-10 20:00
```

После обновления выполнить:

```bash
npm install
npm run deploy
npm run db:migrate
npm start
```
