# ServerCore v14 — расширенная веб-панель

## Что добавлено

- Страница «Участники» с переходом в карточку пользователя.
- Управление пользователем из веб-панели: XP, монеты, репутация, сезонный XP, уровень, сообщения.
- Сброс cooldown `/daily` для конкретного участника.
- Выдача кастомного достижения/бейджа из веб-панели.
- Страница «Экономика» для массовой выдачи/списания XP и монет.
- Страница «Магазин» для добавления, обновления и удаления товаров.
- `/shop` и `/buy` теперь используют товары из базы, если они созданы в веб-панели.
- Страница «Тикеты» с закрытием/открытием тикета из веб-панели.
- Страница «Модерация» со снятием предупреждений.
- Страница «Ивенты» с открытием/закрытием ивентов.
- Страница «Кланы» с управлением банком клана.
- Страница «Заявки» с принятием/отклонением заявок.
- Страница «Интеграции» с добавлением и включением/выключением заготовок.
- Страница «Логи» с быстрым просмотром предупреждений, AutoMod и статуса базы.
- Настройки XP, daily и названий каналов вынесены в отдельные формы.

## Запуск после обновления

```bash
npm install
npm run deploy
npm run db:migrate
npm start
```

Если структура сервера уже создана, `npm run setup` не требуется.

## Веб-панель

Открой:

```text
http://localhost:3000
```

Пароль берется из `.env`:

```env
WEB_PASSWORD=твой_пароль
WEB_SESSION_TOKEN=любая_длинная_строка
```
