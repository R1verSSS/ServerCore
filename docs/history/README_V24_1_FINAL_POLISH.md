# v24.1 Final UX Polish

Это обновление не добавляет новый крупный модуль, а улучшает удобство текущей v24.1.

## Что улучшено

1. Главное меню `/menu` стало более понятным и разделено на пользовательские сценарии.
2. `/help` стал интерактивным: категории выбираются через select-menu.
3. Добавлен онбординг для новых участников через `guildMemberAdd`.
4. Voice-панель стала информативнее: владелец, статус, лимит, участники, подсказки.
5. При создании временной voice-комнаты панель управления остается в чате комнаты.
6. Добавлен единый сервис embed-ответов: success/error/warning/info.
7. Ошибки доступа стали понятнее для пользователя.
8. `/clear` получил подтверждение для удаления больше 20 сообщений.
9. `/dbstatus` теперь показывает health-check проекта.
10. Веб-панель получила страницу `/health`.
11. В веб-панели у настроек появились русские пояснения.
12. Улучшена карточка `/profilecard`: блоки статистики, BP, прогресс, описание.
13. Ответы мини-игр стали информативнее.
14. Логи стали более структурированными.
15. `npm run setup` публикует больше стартовых панелей в нужные каналы.

## Как обновиться

```bash
Ctrl + C
npm install
npm run deploy
npm run db:migrate
npm start
```

Если нужно заново опубликовать стартовые панели:

```bash
npm run setup
```

## Важный момент

Для приветствия новых участников нужен включенный Server Members Intent:

Discord Developer Portal → Bot → Privileged Gateway Intents → Server Members Intent.

Для AutoMod уже нужен Message Content Intent.
