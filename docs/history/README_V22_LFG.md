# v22 LFG / Поиск команды

Добавлен модуль поиска команды для игровых активностей.

## Команды

```text
/lfg create game:CS2 players:5 description:Ищем игроков voice:true
/lfg list
/lfg info id:1
/lfg join id:1
/lfg leave id:1
/lfg close id:1 reason:Команда собрана
```

## Что делает модуль

- создает карточку поиска команды в канале `🎮・поиск-команды`;
- добавляет кнопки `Присоединиться`, `Выйти`, `Закрыть`;
- хранит LFG-записи в базе;
- по желанию создает временную voice-комнату;
- удаляет созданную voice-комнату после закрытия LFG;
- логирует создание и закрытие в `🛡・лог-модерации`.

## Права

Для создания voice-комнат боту нужны права:

```text
Manage Channels
View Channels
Connect
Move Members — желательно для владельца комнаты
```
