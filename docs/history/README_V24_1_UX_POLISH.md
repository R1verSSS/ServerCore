# v24.1 UX Polish

Обновление не добавляет отдельный крупный модуль, а улучшает удобство уже существующей v24.

## Что улучшено

- Добавлена команда `/menu` для удобной навигации по уже существующим функциям бота.
- Добавлена публикация меню через `/menu post` для канала навигации или любого другого канала.
- В `/help` добавлены новые подсказки по меню.
- В `npm run setup` меню сервера отправляется в канал `🧭・навигация`.
- При создании временной voice-комнаты бот отправляет в чат этой комнаты панель управления.
- Панель voice-комнаты теперь позволяет:
  - закрыть комнату;
  - открыть комнату;
  - переименовать комнату через форму;
  - изменить лимит участников через форму;
  - пригласить участника через select-menu;
  - посмотреть информацию;
  - забрать комнату;
  - удалить комнату.

## Как обновиться

```bash
Ctrl + C
npm install
npm run deploy
npm run db:migrate
npm start
```

`npm run setup` запускать необязательно, но если нужно автоматически добавить меню в `🧭・навигация`, можно запустить:

```bash
npm run setup
```

## Проверка

```text
/menu open
/menu post channel:#🧭・навигация
/voice setup
```

После этого зайдите в `➕・создать-комнату`. Бот создаст личную voice-комнату и отправит панель управления прямо в чат созданной комнаты.
