# v15 — кастомизация профиля

Добавлено оформление профиля пользователя:

- `/profilecustomize view` — посмотреть текущие настройки.
- `/profilecustomize title` — изменить титул.
- `/profilecustomize about` — изменить описание.
- `/profilecustomize color` — выбрать цвет профиля.
- `/profilecustomize background` — выбрать фон PNG-карточки.
- `/profilecustomize badge` — выбрать главный бейдж.
- `/profilecustomize visibility` — скрыть/показать бейджи и статистику.
- `/profilecustomize reset` — сбросить оформление.
- `/cosmetics` — посмотреть купленную косметику.

`/profile` и `/profilecard` теперь используют выбранный цвет, фон, титул, описание и главный бейдж.

## Магазин косметики

В стандартный магазин добавлены товары:

- `profile_color_gold` — золотой цвет профиля;
- `profile_color_purple` — фиолетовый цвет профиля;
- `profile_bg_neon` — неоновый фон;
- `profile_bg_ocean` — океанский фон.

Косметику можно покупать через `/buy` или добавлять через веб-панель в разделе «Магазин».

## Веб-панель

Добавлен раздел «Профили». В карточке пользователя можно редактировать титул, описание, цвет, фон и главный бейдж.
