# v23 — Temporary Voice Rooms

## Что добавлено

- `➕・создать-комнату` — голосовой канал-триггер.
- При входе в этот канал бот создает личную voice-комнату и переносит туда пользователя.
- Комната автоматически удаляется, когда становится пустой.
- `/voice setup` — создать/проверить канал-триггер.
- `/voice panel` — отправить панель управления voice-комнатами.
- `/voice lock` — закрыть комнату.
- `/voice unlock` — открыть комнату.
- `/voice limit count` — установить лимит участников.
- `/voice rename name` — переименовать комнату.
- `/voice invite user` — разрешить участнику вход в закрытую комнату.
- `/voice claim` — забрать комнату, если владелец вышел.
- `/voice delete` — удалить свою комнату.
- `/voice list` — список активных временных комнат.
- Веб-панель `/voice` — активные комнаты, создание канала и отправка панели.

## Важно

Для работы нужен intent `GuildVoiceStates`, он уже добавлен в код.
Боту нужны права:

- Manage Channels / Управление каналами
- Move Members / Перемещение участников
- View Channel / Просмотр каналов
- Connect / Подключаться

Если пользователь заходит в `➕・создать-комнату`, но комната не создается, проверь роль бота и права в категории `🔊 ГОЛОСОВЫЕ`.
