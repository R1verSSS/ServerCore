# v21 — Advanced Moderation

Добавлены расширенные инструменты модерации:

- `/cases` — история наказаний участника или последние дела сервера.
- `/case info` — просмотр дела по ID.
- `/case close` — закрытие дела с комментарием.
- `/warnremove` — снятие предупреждения по ID.
- `/note add` — модераторская заметка по участнику.
- `/note list` — список заметок участника.
- `/appeal create` — апелляция пользователя по наказанию.
- `/appeal list` — список апелляций для модераторов.
- `/appeal accept` / `/appeal deny` — решение по апелляции.

Дела автоматически создаются при командах:

- `/warn`
- `/mute`
- `/unmute`
- `/kick`
- `/ban`

В веб-панели раздел `Модерация` теперь показывает:

- дела модерации;
- активные предупреждения;
- модераторские заметки;
- апелляции;
- открытые тикеты.

## Обновление

```bash
npm install
npm run deploy
npm run db:migrate
npm start
```

`npm run setup` запускать не обязательно.
